Part 3. Customising a post

by Danielle Navarro, 14 Apr 2019



Here is some text that I have written. This page uses the light theme. To make life a little easier I’ve also written a small package that supplies ggplot2 themes that are consistent with the slum styles:

#devtools::install_github("djnavarro/slumstyle")
library(slumstyle)
library(tidyverse)

The background colour for this theme is very slightly grey rather than white, so if you want a plot to blend in with the background you need to do something like this:

ggplot(mpg, aes(displ, hwy)) + 
  geom_point() + theme_slum("light")

The CSS file

If you ever feel the need to do so, it’s not too difficult to create a new colour scheme. The static/css/colour_dark.css file specifies the four main colours used:

:root {
  --primary: #ddd;
  --subtle: #888;
  --behind: #333;
  --highlight: hotpink;
}
